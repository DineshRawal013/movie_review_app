# DEF-007 — Trailer URL not extracted from TMDB response; not stored or displayed (FR-027)

**Status:** OPEN

---

## Requirement Violated

- FR-027 (P1) — Detail page links to YouTube trailer if available.
- TC-MOVIES-017: Navigate to detail page of movie with trailer; verify "Watch Trailer" link or play button visible.

---

## Description

`backend/src/tmdb/tmdb.service.ts` fetches movie details with `append_to_response: 'credits,videos'`, so the TMDB response does include video data. However, `backend/src/movies/movies.service.ts` `syncMovieFromTmdb()` does not extract or persist `trailerUrl` from the `videos.results` array.

`frontend/components/movies/movie-detail.tsx` does not render a trailer link at all.

This is related to DEF-006 (same root cause in `syncMovieFromTmdb`) but tracked separately because it is a different field and a P1 requirement.

Note: P3 severity is assigned because FR-027 is P1 but the feature is entirely absent rather than incorrectly implemented, and the fix is co-located with the DEF-006 fix (same method). Reassign to P2 if stakeholder considers missing trailer links a significant regression.

---

## Steps to Reproduce

1. Query `GET /api/movies/550` for Fight Club (which has a trailer on TMDB).
2. Observe: No `trailerUrl` field in the response.
3. Navigate to `/movies/550` in browser: No "Watch Trailer" button/link.

---

## Expected Result

- `GET /api/movies/550` response includes: `"trailerUrl": "https://www.youtube.com/watch?v=SUXWAEX2jlg"` (or equivalent).
- Movie detail page shows "Watch Trailer" button/link when `trailerUrl` is non-null.
- Movie detail page hides trailer element when `trailerUrl` is null.

---

## Actual Result

- `trailerUrl` not present in API response.
- No trailer link on frontend.

---

## Fix Guidance

See DEF-006 backend fix guidance for extracting `trailerUrl` from `tmdbData.videos.results`.

Frontend addition to `movie-detail.tsx`:
```tsx
{movie.trailerUrl && (
  <a
    href={movie.trailerUrl}
    target="_blank"
    rel="noopener noreferrer"
    className="inline-flex items-center gap-2 text-sm text-primary hover:underline"
  >
    Watch Trailer
  </a>
)}
```
