# DEF-008 — MovieCard missing release year display (FR-002)

**Defect ID:** DEF-008
**Severity:** P3 — Minor
**Status:** FIXED

**Phase Detected:** 7 — Test (Static analysis / frontend component review)
**Assigned to:** frontend-engineer

---

## Requirement Violated

- FR-002 (P0) — Browse cards show poster, title, **release year**, avg community rating.
- TC-MOVIES-002 (step 5): Inspect movie card: verify poster image visible, title text visible, **release year visible**.

---

## Description

`frontend/components/movies/movie-card.tsx` renders:
- Poster image (with alt text)
- Title (H3)
- Average rating (star icon + number)
- Rating count

It does NOT render the **release year**. The `movie.releaseDate` field is available in the API response (it is part of the `Movie` type from `lib/api.ts`) but is not extracted or displayed in the card component.

---

## Steps to Reproduce

1. Navigate to home page or browse/search page.
2. Inspect any movie card.
3. Observe: No release year displayed on the card.

---

## Expected Result

Movie card displays: Poster | Title | Release Year | Community Rating
Example: "Fight Club · 1999 · 4.3 ★"

---

## Actual Result

Movie card displays: Poster | Title | Community Rating (year absent)

---

## Fix Guidance

In `frontend/components/movies/movie-card.tsx`, add release year display:
```tsx
import { extractYear } from '@/lib/utils';  // or inline: new Date(movie.releaseDate).getFullYear()

// In the card body, after title:
<div className="flex items-center gap-1 text-xs text-muted-foreground">
  <span>{movie.releaseDate ? new Date(movie.releaseDate).getFullYear() : ''}</span>
  <span>·</span>
  <Star className="h-3 w-3 fill-star text-star" aria-hidden="true" />
  <span>{movie.avgRating > 0 ? movie.avgRating.toFixed(1) : 'N/A'}</span>
  ...
</div>
```
