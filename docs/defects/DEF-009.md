## Requirement Violated

- FR-070 (P0) — Admin can view moderation queue (all, flagged, recent).
- TC-ADMIN-001 (step 1–3): `GET /api/admin/reviews` returns ALL non-deleted reviews (active + hidden), including unflagged ones.
- TC-ADMIN-001 (step 4): `GET /api/admin/reviews?filter=flagged` returns only reviews with `flag_count > 0`.

---

## Description

`backend/src/admin/admin.service.ts` provides only `listFlaggedReviews()` — no method exists to list ALL reviews for the admin queue. The `admin.controller.ts` exposes only `GET /admin/reviews/flagged`:

```typescript
@Get('reviews/flagged')
listFlaggedReviews(@Query() query: PaginationQueryDto) {
  return this.adminService.listFlaggedReviews(query.page, query.limit);
}
```

The api-spec.yaml and FR-070 require `GET /admin/reviews` to return the full moderation queue (all non-deleted reviews regardless of flag count), with an optional `?filter=flagged` query parameter.

The current implementation:
1. Has no `GET /admin/reviews` endpoint (returns 404).
2. Only exposes `GET /admin/reviews/flagged` (partially satisfies `?filter=flagged`).

FR-070 specifies viewing "all, flagged, recent" — all three views are currently missing or incomplete.

---

## Steps to Reproduce

1. Send `GET /api/admin/reviews` with admin auth.
2. Observe: HTTP 404.
3. Send `GET /api/admin/reviews/flagged` with admin auth.
4. Observe: HTTP 200, but only flagged reviews returned (no unflagged active/hidden reviews).

---

## Expected Result

- `GET /api/admin/reviews` returns all non-deleted reviews (active + hidden).
- `GET /api/admin/reviews?filter=flagged` returns only reviews with flag_count > 0.
- `GET /api/admin/reviews?filter=recent` returns most recently created reviews.

---

## Actual Result

- `GET /api/admin/reviews` returns HTTP 404.
- `GET /api/admin/reviews/flagged` returns flagged reviews only.

---

## Fix Guidance

In `admin.service.ts`, add `listAllReviews()`:
```typescript
async listAllReviews(page: number, limit: number, filter?: string) {
  const where: any = { deletedAt: null };
  if (filter === 'flagged') where.flagCount = { gt: 0 };
  const orderBy = filter === 'recent' ? { createdAt: 'desc' as const } : { flagCount: 'desc' as const };

  const [data, total] = await Promise.all([
    this.prisma.review.findMany({
      where, skip: (page - 1) * limit, take: limit, orderBy,
      include: {
        user: { select: { id: true, displayName: true, email: true } },
        flags: { select: { userId: true, createdAt: true } },
      },
    }),
    this.prisma.review.count({ where }),
  ]);
  return { data, meta: buildPaginationMeta(total, page, limit) };
}
```

In `admin.controller.ts`, add:
```typescript
@Get('reviews')
listReviews(@Query() query: PaginationQueryDto, @Query('filter') filter?: string) {
  return this.adminService.listAllReviews(query.page, query.limit, filter);
}
```
